1. What Was Built

An Export button has been added to the top-right of the Bid Queue page header. Clicking it opens a modal asking whether to export as PDF or Excel.

  • PDF — triggers window.print(). The user picks "Save as PDF" in the browser print dialog. No server call needed.
  • Excel — generates a .xls file entirely in the browser using SpreadsheetML (Microsoft's XML format). No external library, no server call. The file downloads instantly and opens in Excel, Numbers, and Google Sheets.

The export always uses the currently filtered bids — so if the user has filtered to "Healthcare" before clicking Export, the file only contains Healthcare bids.

✅ No backend changes required. Everything runs client-side. No new Python routes, no new npm packages.

3. Files in Detail

New file frontend/src/utils/exportUtils.js Export logic

Contains two exported functions. Both are called by ExportModal.jsx.

exportPDF()

Calls window.print(). The browser opens its print dialog where the user selects "Save as PDF". The @media print CSS (if added) can hide buttons and adjust layout.

export function exportPDF() {
  window.print()
}

exportExcel(bids)

Builds a SpreadsheetML XML string in memory, wraps it in a Blob, creates an object URL, and triggers a download. No npm package, no server round-trip.

The file has 4 sheets:

Sheet 1 — Summary
KPI row (Total Bids, Avg Response, Overdue, At Risk, Awarded) + Status breakdown table + Segment breakdown table
Sheet 2 — All Bids
All visible bids with columns: Customer, Bid ID, Segment, OpCo, Status, Response %, Internal Due, Compliance. Alternating row shading. Red/amber/green colour on response and due date cells.
Sheet 3 — Overdue & At Risk
Only bids due within 7 days or with status "At Risk", sorted by urgency (most overdue first). Same colour coding.
Sheet 4 — Compliance Matrix
Bids as rows, compliance tags (CN, Buy America, PFS, Halal, Kosher, Vegan, SOX) as columns. ✓ in green where the bid has that tag, — in light grey where it does not.

The download filename is auto-generated: Sysco_Bid_Pipeline_Jul2026.xls

New file frontend/src/components/BidQueue/ExportModal.jsx Modal UI component

A focused React component. Receives four props:

PropTypePurpose
openbooleanControls visibility
onClosefunctionCalled when user dismisses
bidsArrayThe bids to export (already filtered)
onToastfunctionShows a toast message after export starts

Behaviour:

  • Renders null when open is false (no DOM cost)
  • Closes on Escape key or clicking the backdrop
  • PDF button: closes modal → waits 200 ms → calls exportPDF() (delay lets the modal disappear before the print dialog opens)
  • Excel button: closes modal → calls exportExcel(bids) immediately
import { exportPDF, exportExcel } from '../../utils/exportUtils'

export default function ExportModal({ open, onClose, bids, onToast }) {
  // Escape key closes modal
  useEffect(() => {
    if (!open) return
    const handler = e => { if (e.key === 'Escape') onClose() }
    window.addEventListener('keydown', handler)
    return () => window.removeEventListener('keydown', handler)
  }, [open, onClose])

  if (!open) return null
  // ... modal JSX with PDF / Excel buttons
}

4. How to Test

Start the app

# Make sure App.jsx imports index_v2
cd frontend
npm run dev

Test PDF export

  1. Open http://localhost:5173
  2. Click Export button (top-right of Bid Queue, next to the date pill)
  3. Modal appears — click PDF
  4. Browser print dialog opens. Select a printer or "Save as PDF"
  5. Toast message: "Print dialog opened — choose Save as PDF"

Test Excel export

  1. Optionally filter to a segment (e.g. Healthcare) — export only exports visible bids
  2. Click Export → Excel
  3. File downloads immediately: Sysco_Bid_Pipeline_Jul2026.xls
  4. Open in Excel / Google Sheets — verify 4 sheets: Summary, All Bids, Overdue and At Risk, Compliance Matrix
  5. Toast message: "Excel downloaded — N bids, 4 sheets"

Test modal behaviour

  • Press Escape while modal is open — modal closes
  • Click the dark backdrop behind the modal — modal closes
  • Click the × button — modal closes
